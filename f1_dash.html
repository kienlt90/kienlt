Ứng Dụng Giả Lập & Telemetry Formula 1

Xem thời gian thực các thông số chặng đua, lốp xe, khoảng cách (gaps), tốc độ tối đa (telemetry) và bản đồ mini sectors của các siêu sao F1. Hệ thống được tích hợp bằng công nghệ Next.js, SignalR và Rust.

HƯỚNG DẪN KHỞI CHẠY

💡 Cách nhanh nhất: Nhấp đúp vào tệp run_f1.bat ở thư mục gốc dự án. Hệ thống sẽ tự động cấu hình môi trường, cài đặt thư viện và khởi chạy máy chủ Next.js cho bạn.
1 Truy cập thư mục Client
Mở terminal tại thư mục chứa source code F1 và di chuyển đến thư mục dashboard: cd f1-dash/dashboard
2 Cài đặt các gói phụ thuộc (Yarn/NPM)
Cài đặt thư viện Next.js và các package cần thiết: yarn install # Hoặc npm install
3 Khởi động server phát triển
Chạy lệnh để Next.js lắng nghe tại cổng 3000: yarn dev # Hoặc npm run dev
🚀 Truy cập Dashboard (Cổng 3000) ➔
