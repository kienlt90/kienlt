Bạn có thể chỉnh sửa trực tiếp đoạn code JS tự sinh để đáp ứng các nghiệp vụ phức tạp hơn:

1. Đổi mã máy sang mã LIS bệnh viện

Đổi các mã mặc định như CHISO_4 thành mã LIS (ví dụ WBC, RBC...):

// Đổi mã trong mảng mappings hoặc results:
{ code: "WBC", line: 4 },
{ code: "RBC", line: 5 }

2. Lọc sạch kết quả (Chỉ giữ lại số)

Dùng Regex để loại bỏ ký tự lạ, giữ lại số và dấu thập phân:

// Chỉ lấy số và dấu chấm thập phân:
var cleanVal = rawVal.replace(/[^\d.-]/g, "");

3. Định vị cờ bất thường hoặc Dương/Âm tính

Cách đọc cờ cảnh báo của máy để đổi thành văn bản LIS:

if (rawVal.indexOf("+") >= 0) return "Dương tính";
if (rawVal.indexOf("-") >= 0) return "Âm tính";

4. Tự động tính chỉ số phụ hoặc gộp dòng

Bạn có thể tính chỉ số mới từ các chỉ số máy gửi về:
